# Reglas
-No tienes que ejecutar siempre pnpm run build cada vez que creas una nueva pantalla
- Debes ejecutarlo para verificar que no hay errores
- No debes ejecutarlo si no es necesario
- No es necesario detener el servidor de desarrollo cada vez que haces un cambio ya que se reinicia solo

# pnpm run dev - Ejecuta el servidor de desarrollo
